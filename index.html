Projects

What I've built.

Featured capstone project from my senior year at Wichita State University.

🍯
Honey Sentinel
Senior Capstone Project • Spring 2026

IoT-Based Beehive Monitoring System

Honey Sentinel is an IoT system designed to help beekeepers track hive health remotely. The system uses a Raspberry Pi with environmental sensors to collect real-time temperature, humidity, and weight data, displayed through a web-based dashboard for remote monitoring.

As a core developer on a four-person team, I was responsible for system architecture, network design, deployment documentation, and testing. The project includes comprehensive documentation covering architecture, deployment guides, network design, test plans, isolation standards, operator runbooks, and data dictionaries.
